## How it works

=> input = UART ?
=> esc = disable TX (auto disable when message stop)
=> letters are sent when typed, buffer of only a few characters
=> when going from letter to number or opposite => append to buffer
=> BUFFER_CLEAR and BUFFER_FULL leds, indicating if you can type

=> outputs UART (or SPI, will see)

flow:
keyboard => circular buf (put char or 0x00 at each RTTY sub-clk) => => TX (FIFO) at each subclk
=> at each CLK go to next buf => if last = end TX


? => for circular buf use "shift reg" style instead of usual soft
=> reg [?:0] => usage => if usage > size => stop putting stuff
=> when pull, usage-- && shift
=> when push, if usage < max => ++ && shift
